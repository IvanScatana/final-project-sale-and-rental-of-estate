# Анализ рынка недвижимости: Разведочный анализ данных (EDA)

## Описание проекта


Данный проект выполнен командой seaborn_team в следущем составе:
- Бек https://github.com/beeek18
- Ваня https://github.com/IvanScatana
  
Посвящен разведочному анализу данных (EDA) о рынке аренды в Москве. Цель исследования — выявить основные закономерности, тренды и аномалии в данных, а также подготовить их для дальнейшего моделирования.

**Основные задачи EDA:**
- Очистка данных от пропусков и выбросов
- Анализ распределения ключевых признаков
- Исследование взаимосвязей между характеристиками объектов
- Выявление факторов, влияющих на стоимость недвижимости
- Подготовка данных для построения предсказательных моделей

**Описание подготовки данных к анализу**
- Парсинг из текстовых полей: цена за месяц (руб.), площадь (м²), этаж, количество комнат.
- Категория этажа: низкий (1–3), средний (4–9), высокий (10+).
- Фильтрация выбросов по перцентилям 1% и 99% для цены и площади. Исходно 22 303 записей → 21 542 после очистки (удалено 761, 3.4%). Границы: цена 19 000–600 000 руб., площадь 22–238 м².

## Ключевые выводы
- **Цена и площадь** — сильная положительная связь (корреляция 0.825 после фильтрации); после удаления выбросов связь усилилась (+0.056).
- **Цена и число комнат** — средняя положительная связь (0.629); **площадь и комнаты** — сильная (0.819).
- **Этаж** слабо связан с ценой (0.104); для моделирования малополезен.
- После фильтрации: медианная цена 50 000 руб/мес, медианная площадь 50 м², в среднем ~2 комнаты.
- Распределение объявлений по комнатам: максимум у 1–2-комнатных; данные готовы к построению предсказательных моделей.
